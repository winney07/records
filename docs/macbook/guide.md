#### Mac连接window 文件

1. 连在同一Wi-Fi下 

2. 先将磁盘属性设置为共享

3. 查看windows电脑ip 地址：

   ```
   windows键 + R 
   ```

4. 输入`ipconfig` 获取ip地址（IPv4地址）

5. Mac电脑菜单栏中选择“前往”——“连接服务器”（快捷键：command+ K)，

   输入smb://+windows电脑ip 地址：

   ```
   smb://192.168.**.**
   ```

#### 8.自定义电话铃声、闹钟铃声

1. `下载“库乐队”`
2. 网易云音乐—选中音乐—音乐`右侧的...按钮`—“设置铃声或彩铃”
3. 点击拷贝到`“库乐队”`（等待歌曲上传成功—在“库乐队”的“最近项目中”显示）
4. 选中音乐文件，长按，`选择“共享”`
5. 选择`“电话铃声”`
6. 选择`“导出”`
7. 导出成功后，选择`“设置为铃声”`，或者在手机的`设置里的“声音”中选择音乐文件`
