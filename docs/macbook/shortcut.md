# 快捷键

#### [Mac 键盘快捷键](https://support.apple.com/zh-cn/HT201236)—Apple官网

Command等于Ctrl

Option等于Alt

浏览器缩放比例：

```
command +  "+"  
command +  "-"
```

显示隐藏文件、文件夹，再按一次，恢复隐藏：

```
Command + Shift + . 
```

finder下使用Command+Shift+G 可以前往任何文件夹，包括隐藏文件夹

```
Command + Shift + G 
```
